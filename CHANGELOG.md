# 更新日志

## 1.2.0 - 2026-06-01

- 同步 GitHub 最新 UBAA 签到实现：统一认证后从 iClass MyCenter 跳转解析 `loginName`，不再依赖已失效的学号直登。
- 修复 WebVPN 重定向中 `file/...` 路径可能被识别为不支持 URI scheme 的问题。
- 签到提交对齐 UBAA 最新表单请求方式，并在请求头携带 `sessionId`。
- 直连与 WebVPN 登录页统一使用统一认证密码，WebVPN 额外填写 VPN 账号。

## 1.1.0 - 2026-06-01

- 恢复并修复 WebVPN 登录入口，按 WebVPN 网关规则动态封装 iClass、SSO 与签到接口地址。
- 修复 Android WebVPN 链路可能出现的 `URLConnectionInvoker` 类型转换异常，网络请求改用托管 HTTP 处理器。
- 签到时优先读取学校服务器时间戳，并保留登录阶段 Date 响应头时间同步与 WebVPN 负向修正，降低“参数错误”概率。
- 直连模式继续保留学号输入；当学校接口返回用户不存在时，会尝试通过 iClass 跳转解析登录名后重试。
- 首页新增软件更新检查按钮，支持从 GitHub Release 下载 APK 并打开安装程序。
- 课程、签到信息和二维码生成过程中增加状态提示，减少界面像卡住的情况。
- 首页补充开源使用提醒、隐私说明和签到注意事项。

## 1.0.0 - 2026-06-01

- 新增仓库说明和构建说明。
